# ⚡ Vercel Frontend Deployment

## Instant Deploy (20 seconds)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Vetri1706/Brand-identity-generator&project-name=ai-logo-generator&repository-name=ai-logo-generator&root-directory=frontend)

## Manual Deploy (1 minute)

1. **Go to Vercel**: https://vercel.com
2. **Sign up** with your GitHub account
3. **Click "Import Project"**
4. **Select**: `Vetri1706/Brand-identity-generator`
5. **IMPORTANT**: Set **Root Directory** to `frontend`
6. **Deploy automatically!**

## Configuration

Vercel auto-detects:

- ✅ **Framework**: Next.js
- ✅ **Build Command**: `npm run build`
- ✅ **Output Directory**: `.next`

## Your Frontend URL

After deployment, you'll get:

```
https://ai-logo-generator-vetri1706.vercel.app
```

## Environment Variables

In Vercel dashboard, add:

```
NEXT_PUBLIC_API_URL=https://your-backend-url.railway.app
```
